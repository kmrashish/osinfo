Windows Tips and Tricks Section

  • Some Really Cool and Handy Keyboard shortcuts

    • Window Key + Arrow Keys

      Pressing the Windows key + Left or Windows key + Right arrow keys will dock the window your viewing to the left or right-hand side of the screen. Pressing the Windows key + Up arrow will maximize a window and pressing the Windows key + Down arrow will resize and then minimize the window.

      Use Shift:       To stretch a window vertically, press the Windows key + Shift + Up arrow. If you have multiple monitors use the Windows key + Shift + Left or Right arrow keys to move the window to another monitor.

    • Window Key + plus/minus key

      If you ever need to zoom in or out of what is being displayed on your screen press the Windows key + plus/minus keys. Pressing the plus zooms in, and once zoomed in, the minus will zoom out.

    • Minimizing all background windows

      If you are not viewing a window in full screen and have other windows open in the background, you can minimize all the background windows by pressing the Windows key + Home key.

  • Using Windows Search

    • Window Key + Arrow Keys

      You can search for and run any program using the Windows search box. Press Windows key and type the full or partial name of the program you wish to run or file you wish to edit. Windows will usually do a good job at finding an exact match and simply pressing enter will execute that program or open that file. If more than one match is found, use the up and down arrow keys to select the file you wish to open and then press Enter.

    • Invoking programs in Administrator mode

      Some programs may require Administrator mode in order to function properly. You can run any program as administrator from the search box by typing the name of the program you want to run and then instead of pressing just Enter to run the program press Ctrl + Shift + Enter to Run As Administrator. If you want to click and run a program as administrator hold down the Ctrl + Shift and then click the icon.

  • Make full use of the Task Bar

    • Pin your favourite programs to the Task Bar

      All of your favourite programs can be pinned to the Taskbar so that you can access them without having to use the Start Screen/Menu. To Pin a program to the Taskbar, right-click on the program or shortcut to the program and in the menu choose the Pin to Taskbar option. Alternatively, you can also drag the icon to the Taskbar.
      If you want to remove a pinned program, right-click on the Taskbar icon and click Unpin this program from the Taskbar. Alternatively, you can also drag the icon off from the Task Bar and then click Unpin this program from the Taskbar.

    • Move the pinned program icons freely

      Any pinned or opened program on the Taskbar can be moved and arranged. Left-click on any icon on the Taskbar and drag it to the location you want it to remain.

    • Windows key + Number key

      Pressing the Windows key + a Number on the top row of your keyboard will open the window present at that number on the Task Bar. For instance, if the programs on the Task Bar are in the order: Windows Explorer, Mozilla Firefox, Google Chrome and so on, then pressing Windows + 1, Windows + 2 and Windows + 3 will invoke Explorer, Firefox and Chrome respectively.

    • Pin your favourite folders

      If you frequently access the same folder, pin it to the Windows Explorer icon on the Taskbar. To do this, drag that folder to the Taskbar until you see Pin to Windows Explorer. Once pinned, Right-Click the Explorer icon to access that folder under the Pinned section.

    • Go to the Desktop quickly

      Click on the farthest right edge of the Taskbar to get to the Desktop.
